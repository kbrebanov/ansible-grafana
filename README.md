grafana
=======

Installs and configures Grafana.

Requirements
------------

[![Build Status](https://travis-ci.org/kbrebanov/ansible-grafana.svg?branch=master)](https://travis-ci.org/kbrebanov/ansible-grafana)

This role requires Ansible 1.9 or higher.
